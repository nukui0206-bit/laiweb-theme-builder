# Laiweb Theme Builder

Unified WordPress theme development workspace for reusable Laiweb site templates.

## Structure

```text
docs/                  Project notes and implementation plan
new-theme/lai-template Working WordPress theme
sources/               Local reference copies from existing sites, not tracked
```

## Current Goal

Build a reusable WordPress theme that keeps the existing production workflow flexible while reducing repetitive setup work.

Initial MVP:

- selectable header layouts
- selectable footer layouts
- selectable loading screens
- reusable CTA and visual parts
- `index.php` remains available for high-freedom custom top page work
- `assets/css/add.css` remains available for final project-specific CSS overrides
